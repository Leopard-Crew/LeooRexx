LeooRexx is a curated Leopard-native ooRexx cut.

Reference system:
Mac OS X 10.5.8 Leopard PowerPC.

Principles:
- Cupertino 2009
- no hacks
- no portability priority
- no plugin ecosystem
- native capabilities define the interface
- ooRexx is the engine, Leopard is the system
- later macOS ideas may inspire, but never be backported
